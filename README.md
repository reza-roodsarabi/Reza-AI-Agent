# Reza AI Agent

A Python-based AI Agent for managing freelance projects.

## Features

- Add new projects
- Edit projects
- Delete projects
- Search projects
- Sort projects by budget
- Project statistics
- Dashboard
- Export projects to CSV
- Import projects from CSV
- Favorite projects
- AI project matching

## Technologies

- Python 3
- JSON
- CSV
- Git
- GitHub
